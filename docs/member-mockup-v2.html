viii. ― F.A.Q.

Frequently, quietly askedよくいただくご質問。

i. どこで利用できますか? +
Alpha が事前に提携している会場でのみご利用いただけます。 提携先は、経営者のご利用に相応しい厳選バー・カフェ・ホテル・会員制サロンに限っており、 提携外の場所（自宅・オフィス・一般店舗等）へのご対応はいたしておりません。 これは施工品質・会場様との信頼関係・安全管理を担保するための、Alpha Lounge の根幹となる仕組みです。 現在のエリアは宮崎・都城が中心で、順次拡大予定です。
ii. ニコチンは本当に入っていませんか? +
はい。Alpha Lounge で使用するのはノンニコチンのハーブシーシャのみです。 タバコ葉を含まない設計のため、禁煙中の方、女性、健康に配慮される経営者の方にも安心して同席いただけます。
iii. 火の粉や臭いで、会場を汚すことはありませんか? +
電熱式の機材を採用しているため、炭を熾す必要がありません。 灰・火の粉・焦げ臭さの心配が極めて少なく、ホテルや落ち着いたカフェの空間でも、静かに馴染みます。 会場確認は提携施工者側が事前に行います。
iv. β会員とはなんですか?正式版とどう違いますか? +
β会員は、正式版リリースに先立つ3ヶ月間の限定募集枠で、先着10社に限りご案内しています。 会費は ¥5,500/3ヶ月（正式版は年会費 ¥33,000、法人 ¥55,000）。 Founding Members として、正式版移行後もβ会員価格を継続適用いたします。
v. 審査制とありますが、誰でも申し込めますか? +
申込自体はどなたでも可能ですが、経営者・個人事業主の方を対象として審査を行わせていただきます。 会員様同士の空気を守るための、静かな配慮です。15分の面談を経て、正式にご案内いたします。
vi. 自分の店舗を、提携会場として登録したいのですが。 +
経営者層のお客様を定常的に迎えたい会場様からのお問い合わせを歓迎しております。 フォームの「利用シーン」欄に「会場提携の相談」とご記入のうえ送信ください。 会場の雰囲気、運営形態、Alpha Lounge との相性を個別にご相談させていただきます。
